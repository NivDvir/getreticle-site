# getreticle.com

Source for [getreticle.com](https://getreticle.com) — the site for **Reticle**, an on-screen guidance engine for macOS.

Reticle is built on [GroundingKit](https://github.com/NivDvir/screen-overlay-toolkit): on-device screen-region grounding with Qwen2.5-VL via Apple MLX. All local, Apple Silicon, no cloud.

Deployed via GitHub Pages from `main`.
